# Proyecto-Algebra-Lineal-Avanzada-y-Modelamiento

# Proyecto 2: PageRank sobre redes reales — Twitter (ICWSM)

## Datos
Red obtenida de KONECT: http://konect.cc/networks/munmun_twitter_social/
El notebook descarga y descomprime el dataset automáticamente al ejecutarse
(no es necesario descargarlo manualmente).

## Requisitos
- Python 3.x
- numpy, scipy, networkx, matplotlib

Instalar con:
pip install numpy scipy networkx matplotlib

## Ejecución
1. Abrir `Twitter.ipynb` en Jupyter o Google Colab.
2. Ejecutar todas las celdas en orden (Run All).
3. Las figuras se guardan automáticamente en la misma carpeta
   (distribucion_grados.png, curva_convergencia.png, etc.).

## Informe
El informe final está en `informe_page_rank.pdf`
